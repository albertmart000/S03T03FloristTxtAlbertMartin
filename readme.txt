1. Per començar el programa es demana el nom de la floristeria amb la qual vols treballar.
Si existeix, carrega les dades del fitxer que té el mateix nom que la floristeria i que està guardat a la carpeta "data".
Si es vol treballar amb una base de dades nova, només cal introduir el nom. El programa crea l'arxiu i el guarda a la carpeta "data".

2. L'objecte "Florist" s'ha guardat en format JSON fent servir la llibreria GSON.

3. Per gestionar els canvis en la quantitat d'unitats de l'estoc s'ha implementat l'interface Cloneable en la clase Product.

4. Cada funció s'ha guardat en un clase independent, per intentar que el codi sigui més clar.
L'accés a aquestes funcions es fa des del Menú Principal de la clase App.
