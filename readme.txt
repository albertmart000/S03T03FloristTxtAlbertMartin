1. Per començar, el programa demana el nom de la floristeria amb la qual vols treballar.
Si existeix, carrega les dades del fitxer que té el mateix nom que la floristeria i que està guardat a la carpeta "data" per tal de poder treballar amb una base de dades existent.
Si es vol treballar amb una base de dades nova, només cal introduir el nom. El programa crea l'arxiu i el guarda a la carpeta "data".

2. L'objecte "Florist" s'ha guardat en format JSON fent servir la llibreria GSON

3. Cada funció s'ha guardat en un clase independent, per intentar que el codi sigui més clar.
L'accés a aquestes funcions es fa des del menú principal de la clase App.
